# What this project is
This project is wall mounted with a moving arm raspberry pi PC.This is made because the raspberry pi is easily upgrade able and has lots of USB ports. This idea is made so you can upgrade as you change what your making like soldering plug in a microscope to see better, 3D modeling pull the screen down and model etc.

# Why I made it
I made this because I have the problem of switching things fast so my computer needs to be ever changing. I found that when I buy a laptop it is not always what I need and takes space on the table and is not very upgrade able for new projects but a full PC is not much better and is more expensive. This is also great because even tho not battery powered you can take it off and take it with you to work and plug it in a monitor or TV. It can plug into up to two monitors, screens or tv's. This project is very upgradeable forwhatever you need it for and is under $300 dollars.
